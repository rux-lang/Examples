# FizzBuzz

Print numbers from 1 to 20, replacing:
- multiples of 3 with **Fizz**
- multiples of 5 with **Buzz**
- multiples of both 3 and 5 with **FizzBuzz**

## Concepts

- `for` loop with an inclusive range (`1..=n`)
- `if / else if / else` conditionals
- Modulo operator (`%`)
- Calling a function from `Main`
- Formatted printing with `Print`

## Expected Output

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
Fizz
19
Buzz
```

## Run

```sh
rux build
rux run
```
